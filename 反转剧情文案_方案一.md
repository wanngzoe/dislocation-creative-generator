# 反转剧情文案 - 方案一

## 方向提取 Prompt

```
分析以下内容，提取用于广告文案创作的方向要素。

## 内容
{subtitle_text}

## 输出要求（严格JSON数组）
[{
    "核心冲突": "一句话描述最抓人眼球的核心冲突（如：至亲背叛、身份错位、求助者正是伤害她的人）",
    "意象词": ["广告文案可直接使用的具体物品/场景词，如：馒头、镯子、卖身契、骨头汤等"],
    "情绪基调": "整体情绪氛围（如：绝望、讽刺、悲凉、愤怒）",
    "可用要素": "可用于创作的关键元素提取"
}]

## 提取原则
1. 只输出JSON数组
2. 意象词要具体名词，可用逗号分隔
3. 核心冲突要一句话描述核心冲突，不描述过程细节
4. 如有敏感意象，可用标点符号分隔或用近义词替代

重要：只输出JSON数组，不要任何其他内容！
```

## 生成文案 Prompt

```
根据以下故事要素，写{count}条短视频配音文案。

## 故事要素
{story_material}

## 核心原则
**画面感 + 反转冲击力 + 吸引人点击**

## 创作方法
1. **画面感**：每句话要有极强的画面感和反转冲击力，刀刀致命。直接可用（如：凌晨四点的厨房、婆婆的大脸、500万）
2. **语气有反差**：根据故事要素中的设定，选取最有冲击或反差的语气叙述（如：小女孩：稚嫩又刺骨，儿媳：淡定又爽感）

## 格式要求
- 每条{duration}（10秒约20字，30秒约100字，60秒约200字）
- 直接可用，不要解释

## 输出格式（严格JSON）
[
  {
    "copy": "文案（画面感+反转+吸引人点击）",
    "conflict_angle": "冲突类型",
    "relationship": "关系视角"
  }
]

重要：只输出JSON数组！
```

## 使用说明

1. **输入素材**：粘贴视频原文字幕，或直接输入方向描述
2. **选择方案一**：使用方向提取
3. **分析方向**：AI提取核心冲突、意象词、情绪基调、可用要素
4. **生成文案**：基于提取的方向要素生成多条文案
5. **筛选使用**：选择最符合的文案直接使用

---

## 关键点

- **意象词要具体**：`凌晨四点` 而非 `早上`
- **语气有反差**：淡定叙述，不煽情
- **画面感强**：让人脑海里能浮现画面
- **反转冲击力**：刀刀致命
