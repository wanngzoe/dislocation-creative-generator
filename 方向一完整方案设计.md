# 方向一：完整方案设计文档

## 概述

方向一是当前使用的主要方案，包含两个步骤：
1. **方向提取**：从故事素材中提取创作要素
2. **文案生成**：基于提取的要素生成短剧文案

---

## 方案一：方向提取

### 1.1 方向提取 Prompt（get_direction_prompt_v1）

```python
def get_direction_prompt_v1(subtitle_text):
    return f"""你是一个顶级的爆款故事拆解专家。你的任务是分析输入的故事大纲或文本，提取用于创作极具视觉冲击力与信息缺口的核心要素

## 内容
{subtitle_text}

## 输出要求（严格JSON数组）
[{{
    "主人公": "第一人称主人公身份（如：被卖的小女孩、新婚儿媳、太子、丫鬟等）",
    "核心冲突": "一句话描述最抓人眼球的核心冲突（如：至亲背叛、身份错位、求助者正是伤害她的人）",
    "意象词": ["广告文案可直接使用的具体物品/场景词，如：馒头、镯子、卖身契、骨头汤等"],
    "情绪基调": "整体情绪氛围（如：绝望、讽刺、悲凉、愤怒）",
    "可用要素": "可用于创作的关键元素提取"
}}]

## 提取原则
1. 只输出JSON数组
2. 主人公要明确（如：小女孩、儿媳、太子）
3. 意象词要具体名词，可用逗号分隔
4. 核心冲突要一句话描述核心冲突，不描述过程细节
5. 核心冲突和可用要素中，使用具体的、有画面感的动作词（如：焚烧、凌迟、沸煮），避免只用抽象词（如：惨烈、痛苦）
6. 如有敏感意象，可用标点符号分隔或用近义词替代

重要：只输出JSON数组，不要任何其他内容！"""
```

### 1.2 方向提取输出字段

| 字段 | 说明 | 示例 |
|-----|------|------|
| 主人公 | 第一人称主人公身份 | 新婚儿媳、太子、丫鬟 |
| 核心冲突 | 一句话描述核心冲突 | 被婆婆凌晨四点叫醒做早饭 |
| 意象词 | 具体物品/场景词（数组） | 行李箱、登机牌、机票 |
| 情绪基调 | 整体情绪氛围 | 人间清醒、极速反转 |
| 可用要素 | 关键元素提取 | 假装买菜直接飞回北京退婚 |

## 方案二：文案生成

### 2.1 文案生成 Prompt（get_plot_twist_prompt_v1）

```python
def get_plot_twist_prompt_v1(story_material, count, word_count, requirements, mode="product"):
    return f"""你是一个拥有千万粉丝的首席爆款文案师。你的任务是根据以下故事素材，写{count}条极具吸引力的短文案（Hook）。

要求：
- 黄金3秒开篇
- 画面感强，反转冲击力大，刀刀致命
- 主人公第一人称视角
- 不要输出任何语气标签、括号注释
- 描述事件时使用动作化表达，避免心理描写和流水账叙述

故事素材：
{story_material}

格式要求：
- 每条{word_count}字
- 直接可用，不要解释

附加要求：
{requirements}

输出JSON数组：
[
  {{
    "copy": "文案",
    "conflict_angle": "冲突角度",
    "relationship": "关系视角"
  }}
]

重要：只输出JSON数组！"""
```

### 2.2 文案生成输出字段

| 字段 | 说明 | 示例 |
|-----|------|------|
| copy | 文案内容 | 新婚第二天被婆婆叫醒做饭，我直接买机票回北京 |
| conflict_angle | 冲突角度 | 反抗传统规训 |
| relationship | 关系视角 | 婆媳冲突 |

### 2.3 文案质量要求

1. **黄金3秒开篇**：开头就要抓住注意力
2. **画面感强**：让读者能想象出画面
3. **反转冲击力大**：结尾要有爆点
4. **刀刀致命**：每句话都要有力度
5. **第一人称视角**：使用"我"来叙述
6. **动作化表达**：避免心理描写，用动作说话
7. **避免流水账**：不要平铺直叙

## 工作流程

```
┌─────────────────┐
│   输入故事素材    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  步骤1: 方向提取  │  ◀── 选择"方案1：方向提取"或"方案2：故事总结"
│  (方向提取 Prompt) │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  提取的要素      │  ◀── 可编辑调整
│  主人公/核心冲突 │
│  意象词/情绪基调 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  步骤2: 文案生成  │  ◀── 输入数量、字数、附加要求
│  (文案生成 Prompt) │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  生成的文案      │
│  copy + conflict │
│  + relationship   │
└─────────────────┘
```

## 直接输入模式

除了通过方向提取生成文案，还支持直接输入故事要素：

```python
# 用户可勾选"直接输入故事要素"，跳过方向提取步骤
# 直接在文本框中输入格式如下的内容：

主人公：北京独生女新媳妇
核心冲突：被婆婆凌晨四点叫醒做早饭
意象词：凌晨四点闹钟、行李箱、登机牌
情绪基调：人间清醒、极速反转
可用要素：假装买菜直接飞回北京退婚
```

## 适用素材示例

### 婆媳冲突类
```
主人公：北京独生女新媳妇
核心冲突：新婚次日被婆婆凌晨四点叫醒做全家早饭
意象词：凌晨四点闹钟、行李箱、登机牌、围裙
情绪基调：人间清醒、极速反转、讽刺爽快
可用要素：阶级降级打击、传统糟粕立规矩、假装顺从实则跑路
```

### 复仇类
```
主人公：隐忍复仇的寡妇
核心冲突：丈夫被渣男王爷的恶毒小妾活活烤死，她冒死恢复处子身潜入京城复仇
意象词：羊汤摊、烈火、药瓶、首饰、剪刀、处子红
情绪基调：隐忍、愤怒、复仇、爽感
可用要素：身份反差、极端暴力场景（活烤丈夫）、身体改造（恢复处子身）
```
